
$ git init<=this command used to initialise the git repo (local)

$ git status<=to check status

$ git status

$ git add index.txt<=to save file or commit it into the local git

$ git status


$ git commit -m "Added new file indx,txt" <= used to commit actions

$ git status
$ git remote add origin https://github.com/SohamGokhale952/github-devops.git<=to add the file in to the remote repo(GitHub)
